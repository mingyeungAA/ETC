# 🍞JSP안에 JAVA

### 🍰<%@ %> 지시태그

>  페이지의 인코딩이나 사용언어를 설정하거나 다른 html, jsp파일의 내용을 삽입하는 내용.

- <%@ page  %>
- <%@ import %>
- <%@ include file="URL" %>

<br>

### 🍰<%! %> 선언태그

> 멤버변수나 메서드를 선언할때 사용 스크립트릿 태그인 <% %> 안에 선언하게 되면 **지역변수**로 선언.

<br>

### 🍰<% %> 스크립트릿 태그

> html코드 안에 자바코드를 사용할 수 있게 해주는 태그.
